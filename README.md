# Censorship or Legal Compliance?

## Executive Summary
This repository contains and tracks the materials for a CMSC 33260 course project about government and legal requests sent to X (formerly Twitter). The project focuses on two types of requests: removing or withholding content and disclosing user information.

## Workflow
The planned analysis uses public X transparency reports for a small descriptive comparison. Lumen records, internet-freedom reports, and prior research could provide additional context and help identify limitations in X's reporting. 

Current materials include an annotated [source list](sources.md). Data collection and analysis are ongoing, and the final reporting period and comparison design remain subject to data availability.

## Deliverables
The expected deliverables of this project include

-   **A brief writeup** that describes the data source, collection, processing, and analysis.



  
-   **A brief video summarizing the project and main findings** uploaded to YouTube.
